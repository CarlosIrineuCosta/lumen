# Lumen UI Requirements

## Glass Insights
- **Black interface** with white text (no gray backgrounds)
- **Private appreciation system** - counts never public
- **Member cards** show avatar + 3 recent photo thumbnails
- **No vanity metrics** - engagement invisible to reduce social pressure
- **Chronological feeds** - no algorithmic manipulation

## Your Requirements
- Max 3 tags per photo (semantic categories)
- Client-side B&W detection (Canvas API, no AI costs)
- Lateral navigation maintains context (swipe through feed/discover/profile)
- Private comments for connected users only
- Tag validation prevents impossible combinations (Urban+Boudoir)
