# Lumen - Premium Tier Pricing Strategy

## Tier Structure Overview

### Basic - $5/month
- Platform access
- 50 photos/month upload limit
- Basic search (city level)
- Public portfolio

### Standard - $10/month
- Unlimited uploads
- Advanced search filters
- Portfolio customization
- Private galleries
- Basic analytics

### Professional - $50/month
**Target**: Working professionals who value time
- Everything in Standard plus:
- Travel announcements ("I'll be in Paris March 15-20")
- Priority search placement
- Advanced search (equipment, specialties, availability)
- Calendar integration
- Verified badge
- Analytics dashboard (who viewed, search appearances)
- Custom URL (lumen.photo/yourname)
- Portfolio website replacement features

### Agency - $150/month
**Target**: Studios, small agencies, high-end professionals
- Everything in Professional plus:
- 3-5 team accounts
- Contract templates (legally reviewed)
- Automated invoicing
- Private client galleries (password protected)
- API access
- White-label delivery options
- Priority phone/WhatsApp support
- Bulk upload tools

### Founder's Circle - $1,000/year ($100/month)
**Limited to 50-100 members**
- Everything in Agency plus:
- Quarterly strategy calls with founder
- Feature request priority
- Beta access to all features
- Potential equity participation
- Annual founder's retreat
- Legacy pricing forever
- Name in platform credits
- Direct influence on platform direction

## Value Propositions

### Why Professional ($50) Makes Sense
- Model charging $300/hour: One saved hour = 6 months subscription
- Replaces portfolio website ($30-50/month elsewhere)
- Travel coordination saves days of emails/messages
- Verified badge increases booking rate

### Why Agency ($150) Makes Sense  
- Replaces multiple tools (booking, contracts, galleries)
- Inferior agency tools cost $500+/month
- Team coordination features save admin time
- White-label client delivery adds value

### Why Founder's ($1,000) Makes Sense
- Professionals already spend $5-10K/year on platforms
- Actual influence over platform direction
- Networking with other top professionals
- Potential return if platform sells/grows

## Implementation Strategy

### Phase 1: Launch with Basic/Standard/Professional
- Focus on Professional tier value
- Build portfolio replacement features
- Establish quality baseline

### Phase 2: Add Agency Tier
- After 500+ users
- When team features requested
- Contract system ready

### Phase 3: Open Founder's Circle
- By invitation only
- After platform stability proven
- When seeking growth capital

## Pricing Psychology

### Anchor High
- Show Professional tier as "Most Popular"
- Emphasize time/money saved
- Compare to existing tool costs

### Grandfather Rates
- Beta users keep their rates forever
- Creates urgency to join early
- Rewards early supporters

### Annual Discounts
- 2 months free for annual payment
- Improves cash flow
- Reduces churn

## Key Metrics to Track

### Tier Distribution
- Target: 60% Basic, 20% Standard, 15% Professional, 5% Premium
- Monitor migration between tiers
- Track feature usage by tier

### Revenue per User
- Target average: $17/user
- Track by cohort
- Monitor tier upgrade triggers

### Churn by Tier
- Expect: Basic highest, Founder's lowest
- Professional tier should have <5% monthly churn
- Agency tier should have <3% monthly churn
